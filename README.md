## 📖 Overview

**Templates** is a comprehensive UI component library and design system that helps developers build beautiful, responsive, and accessible web applications faster. It includes a growing collection of **reusable components**, **page templates**, and a **complete design system** with dark/light mode, RTL support, and accessibility built-in.

### Why Templates?

- 🚀 **Ship faster** — Stop building from scratch, start with production-ready components
- 🎨 **Beautiful by default** — Carefully designed components with attention to detail
- 📱 **Fully responsive** — Works on all devices from mobile to desktop
- 🌓 **Dark mode ready** — Built-in dark/light theme with smooth transitions
- 🌍 **Multi-language** — Full RTL/LTR support with i18n
- ♿ **Accessibility first** — WCAG 2.1 AA compliant components
- 🔧 **Type-safe** — Full TypeScript support with strict typing
- 🎯 **Zero dependencies** — No external UI libraries (only Tailwind CSS)
